# 🚀 SkillSprint

> 📚 **A centralized platform to discover, bookmark, and track free tech courses from YouTube and other learning resources.**

SkillSprint helps learners find high-quality **free tech courses** in one place. Instead of searching across multiple platforms, users can explore, bookmark, and track their learning journey through an intuitive and responsive interface.

---

## ✨ Features

* 🔍 Search free tech courses in real time
* ▶️ YouTube Data API v3 integration
* 📚 Curated learning resources
* ⭐ Bookmark your favorite courses
* 📈 Track learning progress

  * 📌 Bookmarked
  * ⏳ In Progress
  * ✅ Completed
* 🌙 Dark & ☀️ Light mode
* 📱 Fully responsive design
* ⚡ Fast and intuitive user experience

---

## 🛠️ Tech Stack

* 💛 JavaScript
* 🌐 HTML5
* 🎨 CSS3
* 📺 YouTube Data API v3

---

## 🎯 Project Goal

SkillSprint was built to simplify the learning experience by providing a **single platform** where students and developers can discover, organize, and track free technical courses efficiently.
